## Usage

### Install
```
git clone https://git.wappier.com/wappier/oxidised/rust-onboarding
cd rust-onboarding
```

### Branch
After change the `dev` branch
```
git checkout dev
```



## Projects

### Board Server

In the directory `rust-onboarding` run in the command line:
```
cd board-server
```

### GraphQl Server

In the directory `rust-onboarding` run in the command line:
```
cd graph-ql-server
```
